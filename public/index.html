This an Awesome Tutorial for creating an Awesome and Beautiful looking Listing Website

We'll create a Static Website with CMS (Content Management System) and deploy to Cloud with CI (Continuous Integration) and CD (Continuous Deployment)

Coding Languages and Technologies used: JAMStack (JavaScript, APIs and Markup), JavaScript (NodeJS), HTML, CSS, Git, Eleventy (Static Site Generator), Netlify (as Cloud and Deployment Framework), GitHub (for CI and source / version control), Decap CMS (previously know as Netlify CMS)
